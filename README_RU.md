# Ophthalmology C2 Content Repo v2

Полная база: **12100 учебных единиц**.

Загрузите содержимое этой папки в GitHub repository и включите GitHub Pages.

После публикации используйте ссылку:

```text
https://ВАШ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ/data/ophthalmology_c2_pack_v2.json
```

Эту ссылку нужно вставить в Android-приложении во вкладке **База**.

## Что внутри

- `data/ophthalmology_c2_pack_v2.json` — основная база.
- `manifest.json` — информация о версии.
- `ophthalmology_c2_pack_v2_audit.csv` — таблица для аудита.
- `index.html` — страница для GitHub Pages.

## Примечание по авторским правам

База содержит короткие производные учебные единицы: словосочетания, clinical patterns, academic paper phrases, EBM/statistics phrases. Она не содержит длинных фрагментов и не является копией исходных книг.
